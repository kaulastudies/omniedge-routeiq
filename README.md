# OmniEdge RouteIQ ⚡️

[![Hackathon](https://img.shields.io/badge/AMD_ACT_II-Track_1-ED1C24?style=for-the-badge)](https://lablab.ai/ai-hackathons/amd-developer-hackathon-act-ii)
[![Status](https://img.shields.io/badge/Status-In_Development-3B82F6?style=for-the-badge)](#)

OmniEdge RouteIQ is a Hybrid Token-Efficient Routing Agent built for the **AMD Developer Hackathon ACT II**. 

It serves as an enterprise-grade middleware that intercepts AI requests, evaluates their complexity in real-time, and intelligently routes them between zero-cost local models (Edge) and high-performance remote hardware (Fireworks AI / AMD Cloud).

## 🎯 Primary Objectives
- **Minimize Token Costs:** Route simple tasks to local models.
- **Maximize Accuracy:** Route complex reasoning tasks to AMD-powered cloud models.
- **Enterprise Observability:** Provide a beautiful, real-time telemetry dashboard to track cloud waste savings and routing latency.

## 🏗️ Architecture
- **Frontend:** Next.js 15, Tailwind CSS, Framer Motion
- **Backend Routing:** FastAPI / Python
- **Local Engine:** Ollama 
- **Cloud Engine:** Fireworks AI
