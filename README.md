# OmniEdge RouteIQ

Award-focused hybrid AI routing layer for AMD ACT II Hackathon.

## Locked positioning

OmniEdge RouteIQ intelligently routes user tasks between local inference, cloud inference, or hybrid execution based on privacy, complexity, latency, token cost, and fallback reliability.

The current scaffold contains the production-ready FastAPI backend under `backend/`.
